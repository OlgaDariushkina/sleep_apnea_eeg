# Анализ ЭЭГ для детекции апноэ сна

Данный репозиторий содержит код для магистерской диссертации.

## Содержание
- Загрузка и предобработка EDF-файлов
- Разведывательный анализ (EDA)
- Классификация (Random Forest, XGBoost, SVM, 1D-CNN)
- Статистический анализ

## Зависимости
Установка: `pip install -r requirements.txt`
